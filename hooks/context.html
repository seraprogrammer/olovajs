Olova $context Hook: Sharing Magic Everywhere! 🌈 ​

Hey young coders! Let's learn about the $context hook - it's like having a magical backpack that can share things with all your components!

What is $context? 🎒 ​

Think of context like a magical bubble that can:

  • Share information with many components at once
  • Pass values down without giving them to each component
  • Keep important information available everywhere

How to Use It! 🎮 ​

First, create your magical bubble:

javascript
import { createContext, $context } from "olova";

// Create a magical bubble
const ThemeContext = createContext("light");

Then use it in your components:

function MagicBox() {
  const theme = $context(ThemeContext);

  return (
    <div style={{ background: theme === "light" ? "white" : "black" }}>
      I change with the theme! 
    </div>
  );
}

Fun Examples! 🎨 ​

1. Theme Switcher ​

javascript
// First, create your context
const ThemeContext = createContext("light");

function ThemeSwitcher() {
  const [theme, setTheme] = $state("light");

  return (
    <ThemeContext.Provider value={theme}>
      <div>
        <button onClick={() => setTheme(theme === "light" ? "dark" : "light")}>
          Switch to {theme === "light" ? "🌙" : "☀️"}
        </button>
        <MagicBox />
      </div>
    </ThemeContext.Provider>
  );
}

2. Magic Treasure Box ​

javascript
const TreasureContext = createContext({ gold: 0 });

function TreasureGame() {
  const [treasure, setTreasure] = $state({ gold: 100 });

  return (
    <TreasureContext.Provider value={treasure}>
      <div>
        <h1>Treasure Hunt! 💎</h1>
        <TreasureDisplay />
        <TreasureCollector setTreasure={setTreasure} />
      </div>
    </TreasureContext.Provider>
  );
}

function TreasureDisplay() {
  const treasure = $context(TreasureContext);
  return <h2>Gold: {treasure.gold} 🪙</h2>;
}

When to Use Context? 🤔 ​

Use context when you want to share:

  • Theme colors 🎨
  • User information 👤
  • Language settings 🌍
  • Game scores 🎮
  • Any magical value that many components need!

Cool Things to Try! 🚀 ​

1. Language Switcher ​

javascript
const LanguageContext = createContext("en");

function LanguageApp() {
  const [language, setLanguage] = $state("en");

  return (
    <LanguageContext.Provider value={language}>
      <button onClick={() => setLanguage(language === "en" ? "es" : "en")}>
        Switch to {language === "en" ? "Español 🇪🇸" : "English 🇬🇧"}
      </button>
      <WelcomeMessage />
    </LanguageContext.Provider>
  );
}

function WelcomeMessage() {
  const language = $context(LanguageContext);
  return <h1>{language === "en" ? "Hello!" : "¡Hola!"}</h1>;
}

Remember! ​

  1. Create your context with createContext
  2. Wrap components with .Provider
  3. Use $context to read values
  4. Context is like a magical bubble that shares values!

Tips for Young Wizards 🧙‍♂️ ​

  • Start with simple values
  • Give your contexts meaningful names
  • Remember to provide a default value
  • Keep your magical bubbles organized

Magic Weather App ​

javascript
const WeatherContext = createContext("sunny");

function WeatherApp() {
  const [weather, setWeather] = $state("sunny");

  return (
    <WeatherContext.Provider value={weather}>
      <div>
        <button onClick={() => setWeather("sunny")}>☀️</button>
        <button onClick={() => setWeather("rainy")}>🌧️</button>
        <button onClick={() => setWeather("snowy")}>❄️</button>
        <WeatherDisplay />
      </div>
    </WeatherContext.Provider>
  );
}

function WeatherDisplay() {
  const weather = $context(WeatherContext);
  const emojis = {
    sunny: "☀️ Beautiful day!",
    rainy: "🌧️ Bring an umbrella!",
    snowy: "❄️ Winter wonderland!",
  };
  return <h2>{emojis[weather]}</h2>;
}
