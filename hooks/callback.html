Olova $callback Hook: Remember Your Functions! 🧠 ​

Hey young coders! Let's learn about the $callback hook - it's like having a super memory for your functions!

What is $callback? 🤔 ​

Think of $callback like a special notebook that remembers your functions so they don't change every time your component updates. It's like having a perfect memory for your code!

Basic Usage ​

javascript
import { $callback } from "olova";

function GameController() {
  const handleClick = $callback(() => {
    console.log("Button clicked! 🎮");
  }, []);

  return <button onClick={handleClick}>Click me!</button>;
}

Why Use $callback? ​

Imagine you're playing a video game:

  • You want the "jump" button to always do the same thing
  • You don't want it to change how it works every time the screen updates
  • You want it to be super fast and reliable

That's exactly what $callback does for your functions!

Cool Examples! 🎨 ​

1. Score Counter ​

javascript
function ScoreKeeper() {
  const [score, setScore] = $state(0);

  const addPoint = $callback(() => {
    setScore((s) => s + 1);
  }, []); // Empty array means it never changes!

  return (
    <div>
      <h2>Score: {score} 🏆</h2>
      <button onClick={addPoint}>Add Point!</button>
    </div>
  );
}

2. Magic Name Generator ​

javascript
function NameGenerator() {
  const [name, setName] = $state("");

  const generateName = $callback(() => {
    const names = ["Dragon", "Unicorn", "Phoenix", "Wizard"];
    const random = Math.floor(Math.random() * names.length);
    setName(names[random]);
  }, []); // Function stays the same!

  return (
    <div>
      <h2>Your magical name: {name || "❓"}</h2>
      <button onClick={generateName}>Generate Name! ✨</button>
    </div>
  );
}

Using Dependencies 🔄 ​

Sometimes you want your function to remember new things:

javascript
function ColorMixer() {
  const [color1, setColor1] = $state("red");

  const mixColors = $callback(() => {
    console.log(`Mixing with ${color1}! 🎨`);
  }, [color1]); // Changes when color1 changes

  return (
    <div>
      <button onClick={mixColors}>Mix Colors!</button>
      <button onClick={() => setColor1("blue")}>Change Color 1</button>
    </div>
  );
}

Remember! 🌈 ​

  1. Use $callback when you want to:

    • Keep functions the same between renders
    • Pass functions to child components
    • Make your app faster
    • Prevent unnecessary updates

  2. Dependencies work like a memory list:

    • Empty array [] = Always remember the same function
    • With values [value1, value2] = Remember new function when values change
