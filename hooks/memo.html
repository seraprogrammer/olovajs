Olova $memo Hook: Making Your App Super Fast! 🚀 ​

Hey there, awesome coder! Today we're going to learn about the $memo hook - it's like giving your app a speed boost!

What is $memo? 🤔 ​

Think of $memo as a magical memory box that remembers things so your app doesn't have to calculate them again and again. It's like having a super-smart friend who remembers answers to math problems!

How Does It Work? 🎮 ​

javascript
import { $memo } from "olova";

function MathWizard() {
  const result = $memo(() => {
    return "2 + 2 = 4";
  });

  return <div>{result}</div>;
}

When Should We Use It? 🎯 ​

Use $memo when:

  • You have calculations that take a long time
  • You don't want to repeat the same work
  • You want your app to be super fast!

Fun Examples! 🌈 ​

1. Number Wizard ​

javascript
function NumberWizard({ number }) {
  const doubledNumber = $memo(() => {
    return number * 2;
  }, [number]);

  return <div>Magic Number: {doubledNumber} ✨</div>;

2. Color Magic ​

javascript
function ColorMixer() {
  const [color1, setColor1] = $state("red");
  const [color2, setColor2] = $state("blue");

  const mixedColor = $memo(() => {
    return `Mixed ${color1} and ${color2}!`;
  }, [color1, color2]);

  return (
    <div>
      <h2>{mixedColor}</h2>
      <button onClick={() => setColor1("yellow")}>Change Color 1</button>
      <button onClick={() => setColor2("green")}>Change Color 2</button>
    </div>
  );
}

3. Super Calculator ​

javascript
function SuperCalculator({ numbers }) {
  const total = $memo(() => {
    return numbers.reduce((sum, num) => sum + num, 0);
  }, [numbers]);

  return (
    <div>
      <h2>Total: {total} 🧮</h2>
    </div>
  );
}

Cool Things to Remember! 💡 ​

  1. $memo is like a smart calculator that remembers answers
  2. It only recalculates when its dependencies change
  3. It makes your app faster by avoiding repeat work
  4. It's perfect for expensive calculations

1. Name Scrambler ​

javascript
function NameScrambler() {
  const [name, setName] = $state("");

  const scrambledName = $memo(() => {
    return name
      .split("")
      .sort(() => 0.5 - Math.random())
      .join("");
  }, [name]);

  return (
    <div>
      <input
        value={name}
        onChange={(e) => setName(e.target.value)}
        placeholder="Type your name"
      />
      <p>Magic Name: {scrambledName} </p>
    </div>
  );
}

2. Star Counter ​

javascript
function StarCounter() {
  const [stars, setStars] = $state(1);

  const starString = $memo(() => {
    return "⭐".repeat(stars);
  }, [stars]);

  return (
    <div>
      <button onClick={() => setStars((s) => s + 1)}>Add Star</button>
      <p>{starString}</p>
    </div>
  );
}
